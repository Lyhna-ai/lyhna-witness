What's live vs. preview

  • Live today: the proxy witnesses real tool calls and the witness CLI renders real receipts (see Install). The receipt itself is the product, and it's real.
  • Preview only: this hosted dashboard view. There's no sign-up, no billing, and no live agent telemetry here yet — it's the design of what the beta dashboard will show, built from the same receipt data the CLI already produces.

Private beta opening soon.

Lyhna asserts action-level witnessed truth only — what crossed the tool boundary vs. what the agent claimed. It does not verify outcomes, delivery, correctness, or anything outside the observed tool-call path.
