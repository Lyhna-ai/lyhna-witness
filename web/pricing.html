What a witnessed action is

A witnessed action = one tool call observed at the boundary — work Lyhna can see, record on a receipt, and classify without overclaiming. The criterion is simple: was a tool call observed? — not which verdict it earned.

  • Counts: any tool call Lyhna observes — whatever its verdict. Approved, failed, refused, escalated, or a claimed-vs-actual mismatch: Lyhna did the work of watching it.
  • Doesn't count: a claim with no tool call behind it. If the agent says it emailed the client but no send crossed the boundary, there's nothing to witness — so there's nothing to bill. That absence is exactly why the receipt flags it DO-NOT-SEND.

You're billed for watching tool calls, not for the agent's empty claims.
